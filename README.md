# Memory Flip Card Game 🃏

A fully functional Memory Flip Card Game Android application 
built with Java in Android Studio.

## Features
- Classic Mode — unlimited moves, match all pairs at your own pace
- Challenge Mode — three difficulty levels with fixed move limits
  - Easy: 4 pairs, 8 moves
  - Medium: 6 pairs, 10 moves  
  - Hard: 8 pairs, 12 moves
- Smooth card flip animations
- Move counter and pairs tracker
- Win and lose dialogs
- Vibrant pink theme

## Tech Stack
- Language: Java
- Framework: Android SDK
- Min SDK: API 24
- UI: RecyclerView, GridLayoutManager, CardView

## How to Run
1. Clone the repo
2. Open in Android Studio
3. Run on emulator or physical device
